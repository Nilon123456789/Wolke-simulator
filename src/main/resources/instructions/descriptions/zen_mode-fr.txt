Il est possible d'activre le mode Zen à partir du menu "Affichage". Le mode Zen cache tous les composants de l'interface pour une expérience de simulation plus immersive. Il permet de créer des zones de haute pression avec un cliqué-glissé gauche et de rajouter de la fumée avec un cliqué-glissé droit, afin de pouvoir intéragir avec la simulation de manière amusante. Il est possible de le quitter avec le bouton "Minimiser" en bas à droite de la fenêtre.
