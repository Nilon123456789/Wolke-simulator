Dans le fichier de constantes dans le dossier wolke dans votre dossier d'utilisateur, il est possible d'activer le mode expérimental démontrant la possibilité du rendu de la simulation avec OpenGL, en changeant la valeur de la constant "USE_OPENGL_ à "true". Notez que cette fonctionnalité est très instable et n'est pas recommandée pour une utilisation en production, où le rendu standard avec Swing est recommandé. Ce mode reste disponible afin démontrer le travail de recherche et d'apprentissage qui a été fait pour se familiariser avec les technologies de rendu sur le GPU avec OpenGL. Consulter le "README.md" du dossier "backend/models/renderer" pour voir les particuliarité d'exécution de ce mode.
